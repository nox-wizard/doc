Get latest sources using Cvs

You can get latest NoX-Wizard sources through anonymous CVS access:

CVS Parameters
Server cvs.sourceforge.net:/cvsroot/noxwizard
User anonymous
Password [press enter]
Method pserver
Module name See the next table.

Quick commands to download sources from console: (if you use wincvs, you must give those commands once you are in the wincvs directory)

cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/noxwizard login
Just press enter as password
cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/noxwizard co modulename

Module Description
src Sources of NoX-Wizard
scripts Scripts for NoX-Wizard
doc Documentation of NoX-Wizard
unicorn Unicorn Suite's Sources
nxwtoolset NoX-Wizard Toolset for Linux sources

If you use Windows you will need WinCVS or Tortoise CVS for get the sources.

WEB

You can also explore sources in cvs using the web by going to http://cvs.sourceforge.net/cgi-bin/viewcvs.cgi/noxwizard/
