cmd_area (  const  chr  ) 
 

Author:
Fax
syntax: 'area [R] [include] [duration] command params:
  • R: the area radius (default = VISRANGE)
  • include:
    • "itm": include items only
    • "chr": include chr only (default)
    • "all": include all
    • duration: number of commands that will be affected from the area effect. (default = unlimited (0xFFFFFFFF))

    Calling 'area with no parameters will result in a default area to be set, centered at the character's position and with the values specified in the script.
    If the character already had a command area set, calling 'area with no parameters will only delete the area, and no new area is set.
