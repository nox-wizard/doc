cmd_add (  const  chr  ) 
 

Author:
Fax
syntax: 'add [ID/scriptID][amount] command params:
  • ID/scriptID: scriptID ($item_.../$npc_...) of the object, or ID of the item as exadecimal or integer number.
  • amount: how much objects you want (default = XSS value for items, 1 for npcs)

If no params are specified the add menu is opened.
You can also pass a generic word as parameter, in that case you will be prompted to choose between a list ofitems whose define is of the type: $item_<name>_<subtype>_<part>.
For example:
'add forge
will result in a menu where you will be able to choose between:
$item_forge1
$item_forge2_1_1
$item_forge2_1_2
$item_forge2_1_3
$item_forge2_1_4
$item_forge2_1_5
$item_forge2_2_1
$item_forge2_2_2
$item_forge2_2_3
$item_forge2_2_4
$item_forge2_2_5

'add stone_chair
will result in a menu where you will be able to choose between:
$item_stone_chair_1
$item_stone_chair_2
$item_stone_chair_3
$item_stone_chair_4
