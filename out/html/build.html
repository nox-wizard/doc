Building your own NoX-Wizard

NoxWizard can be compiled with:

GCC/G++ [Linux/BSD]

You can build NoX-Wizard under Linux with GCC/G++ versions 2.95.x A makefile is included to build sources properly.

Visual C++ [Windows]

Under Visual C++ 6.0 or later, you should set these options to correctly build NoX-Wizard: VC++ 6.0 Options

Project/Settings.. What do
C++/General/Project Options Add the /J option
C++/Code Generation/Use runtime library Select one of the multithreaded libraries
Link/General/Object-library modules Add wsock32.lib and comctl32.lib

Also you need STLport to correctly build your program under Visual C++ platform, due to many problems of the standard STL implementation of Visual C++ compiler. There is also an Stlport Installation on MS Visual C++ 6 Tutorial written by Horian

Borland C++ Builder [Windows]

You can build NoX-Wizard (from cvs) under Windows also with Borland C++ Builder 6 and later. You don't need STLport because it's the default STL library for this version of Borland compiler.

The makefile for BCB6 is named nox-bcb6.mak. For compile, open a command prompt to the sources directory and type make -f nox-bcb6.mak.
