cmd_hide (  const  chr  ) 
 

Author:
Fax
syntax: 'hiding ["on"/"off"][mode]["target"] command params:
    • "on": hides character
    • "off": unhides character
  • mode
    • "skill": hides by skill
    • "spell": hides by spell
    • "perma": permanantly hidden (default)
  • "target": pass this paramter if you want to bypass the area effect

If area effect is active, all characters in area will be hidden. If no area effect is active, or if you pass "target", a target will appear and only the targetted char will be hidden. Passing no paramteres will toggle the charcater's hiding staus between permahidden/unhidden
