Author:

syntax: 'damage [amount]["hp"/"stam"/"mana"] command params:
  • amount: hoe much damage you want to do
    • "hp": decrease HP (default)
    • "stam": decrease stamina
    • "mana": decrease mana

If area effect is active, all characters in area will be damaged. If no area effect is active, or if you pass "target", a target will appear and only the targetted char will be damaged.
