0.82
Parameters:
name[]: the command name
priv: the command privlevel
func[]: the function to be called, pass "" if you want a standard cmd_commandname function to be called
This is an helper function that can be called by scripts to add commands at runtime.
Usually you can use this function like this: 
addCommand("mycommand",mypriv,"");
and cmd_mycommand will be called on command use.

Note:
: this function is a native function if the source coded command system is selected, else this is a normal public function
