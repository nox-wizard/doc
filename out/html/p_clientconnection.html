Client Connection

To connect to a NOX wizard shard you must remove the encryption from the client, this can be done with Ignition or UOrice.
In order to make you server accessible you have to check you IP address set up:
  • open config.cfg
  • in SECTION SHARD set the shard name, this name will appear in the shard selection window when players connect
  • follow the comments and set the IP and PORT of your server, you can leave "localhost" if you are doing local tests.
